Bookmarkable tabs

Tabs that remember where you left them

Each tab syncs to the address bar, so a shared link opens straight to the right one, and the back button steps you through your own trail.

Overview

Four raised beds, one shared hose bib, and a rotation that changes every other season depending on what did well last year.

Watering

Drip lines run every other morning in summer, hand-watered the rest of the year depending on rainfall.

Planting

Direct-sown root vegetables, started seedlings for anything that needs a head start on the season.

Notes

Try the URL hash directly — #watering, #planting, #notes — or use your browser's back button after switching a few tabs.
